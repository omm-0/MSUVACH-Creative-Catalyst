## 📖 About the Project

**MSUVACH Creative Catalysts (MCC)** is a modern, high-performance web platform designed for a creative agency. The website serves as a digital storefront, showcasing the agency's expertise in brand strategy, content creation, and operational systems. 

The site is built as a highly interactive **Single Page Application (SPA)** using pure Vanilla JavaScript, HTML, and CSS. It features dynamic routing, theme switching, and fluid animations without the overhead of external JavaScript frameworks.

### 🎯 Key Highlights
* **"She Builds" Initiative:** A dedicated sub-brand and section highlighting MCC's commitment to female founders, complete with dynamic CSS theme switching (Blue/Gold to Pink).
* **Immersive UI/UX:** Features a custom pre-loader, intelligent scroll-reveal animations, and a customized interactive cursor.
* **Modern Aesthetic:** Deep dark modes, glassmorphism elements, grain overlays, and animated gradient parallax orbs.

## ✨ Features

* **Vanilla SPA Routing:** Hash-based navigation (`#home`, `#about`, `#services`) that mimics a multi-page site with zero page reloads.
* **Dynamic Theme Switching:** Context-aware color palettes that change root CSS variables based on the active route.
* **Intersection Observer Animations:** High-performance, scroll-triggered reveal animations (`fade-up`, `fade-left`, `scale`).
* **Interactive Custom Cursor:** A styled cursor that reacts and morphs when hovering over interactive DOM elements (buttons, links, cards).
* **Animated Data Counters:** Viewport-triggered statistics counters that animate from zero to their target values.
* **Fully Responsive:** Fluid typography (`clamp()`) and modern CSS Grid/Flexbox layouts ensuring perfect rendering across all devices.
* **Custom Accordion FAQ:** Smooth height-transitioning accordions built without external libraries.

## 💻 Tech Stack

* **Structure:** HTML5 (Semantic Markup)
* **Styling:** CSS3 (Custom Properties, Flexbox, Grid, Keyframe Animations, Backdrop-filter)
* **Interactivity:** Vanilla JavaScript (ES6+, DOM Manipulation, Event Listeners, Intersection Observer API)
* **Typography:** [Google Fonts](https://fonts.google.com/) (Alegreya SC) & Custom Local Fonts (Kingred Modern)
